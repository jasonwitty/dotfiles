# prerequisites

FiraCode Nerd Font Mono

```
yay -S ttf-firacode-nerd
```

Starship

```
curl -sS https://starship.rs/install.sh | sh
```

Alacritty

```
pacman -S alacritty
```

Alacritty Themes

```
yay -S alacritty-themes
```

Fish Shell

```
pacman -S fish
```

Pfetch-rs (requires rustup - https://rustup.rs/)

```
cargo install pfetch
```

eza zoxide and fzf

```
pacman -S eza zoxide fzf
```
